# RandomRange-List

A simple C# console application that generates two random integers between 0 and 1000 and iterates through the range between them.

## Technical Overview
- **Language:** C#
- **Logic:** Utilizes the `System.Random` class for number generation and conditional `for` loops to handle range direction.

---

## Teknik Genel Bakış
0 ile 1000 arasında rastgele üretilen iki tam sayı arasındaki aralığı hesaplayan ve listeleyen C# konsol uygulaması.

### Uygulama Detayları
1. **Rastgele Üretim:** `System.Random` sınıfı kullanılarak [0, 1000] aralığında iki bağımsız sayı üretilir.
2. **Mantıksal Kontrol:** Üretilen sayıların büyüklük sıralaması kontrol edilerek döngü başlangıç ve bitiş noktaları dinamik olarak belirlenir.
3. **Yazdırma:** Belirlenen aralıktaki tüm tam sayılar konsol ekranına yazdırılır.

## Kullanım
Projeyi bir C# derleyicisi veya Visual Studio üzerinden derleyip çalıştırabilirsiniz.
